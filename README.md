# 🤖 IBM Machine Learning Professional Certificate - Projects Portfolio

Welcome to my comprehensive repository containing all the core machine learning projects, algorithms, and notebooks developed during my journey through the **IBM Machine Learning Professional Certificate** course. 

The goal of this repository is to consolidate individual machine learning implementations—ranging from foundational regression techniques to complex clustering and recommendation systems—into a structured, production-ready portfolio.

---

## 📂 Repository Structure & Topics Covered

This repository is organized into distinct categories based on the machine learning pipeline and algorithm types:

### 📈 1. Regression Analysis
Foundational algorithms used to predict continuous numeric values, including model evaluation metrics like R-squared, MSE, and MAE.
* **Simple Linear Regression:** Predicting variables using a single feature.
* **Multiple Linear Regression:** Training models on multi-dimensional feature sets (80% train / 20% test splits).
* **Polynomial Regression:** Handling non-linear relationships by transforming features.
* **Non-Linear Regression:** Implementing custom non-linear curves for complex data trends.

### 🎯 2. Classification Models
Supervised learning methods used to categorize data into discrete classes or probabilities.
* **K-Nearest Neighbors (KNN):** Predicting customer categories (`custcat`) for telecommunications profiles.
* **Decision Trees:** Building classification trees to determine the optimal drug treatment for patients.
* **Logistic Regression:** Developing churn models complete with confusion matrices, precision/recall, and log-loss evaluation.
* **Support Vector Machines (SVM):** High-dimensional classification used to detect and classify cell samples as benign or malignant.
* **Ensemble Methods (Random Forest):** Combined with Logistic Regression for robust Heart Attack Risk Modeling, utilizing feature scaling, class weighting, and ROC-AUC comparisons.

### 👥 3. Clustering & Unsupervised Learning
Groupings of unlabeled data points based on feature similarity and structural patterns.
* **K-Means Clustering:** Segmenting customer profiles for targeted data analysis using `scikit-learn`.
* **Hierarchical Clustering:** Implementing Agglomerative Clustering for vehicle dataset analysis using `scipy` and `scikit-learn`.
* **DBSCAN (Density-Based Clustering):** Spatial data density clustering applied to Canadian Weather Station geographical datasets.

### 🎬 4. Recommendation Systems
* **Content-Based Filtering:** Building an engine that leverages item attributes and content features to classify and recommend personalized items to users.
